# Changelog — nexa-json

## [0.2.0] — 2026

### Added

- `Json.read` — read and parse a JSON file
- `Json.write` — write JSON to a file
- `Json.exists` — check file existence
- `Json.delete` — delete a JSON file
- `Json.parse` — parse a JSON string
- `Json.stringify` — serialize to compact JSON
- `Json.pretty` — serialize to indented JSON
- `Json.get` — get value by key
- `Json.set` — set value by key
- `Json.keys` — list all keys
- `Json.values` — list all values
- `Json.create` — create a new JSON file
- `Json.update` — update a key in an existing file
